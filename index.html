Minicurso de Git e GitHub

A importância do versionamendo de código

Sobre

Este minicurso tem como objetivo explicar o que é Sistema de Controle de Versão (Version Control System ou VCS), como funciona o controle de versão ou versionamento de arquivos e sua importância no desenvolvimento de software em equipe. Explicar as vantagens e desvantagens de algumas dos VCS's mais utilizadas atualmente. Também vamos aprender alguns comandos básicos do Git, que é o VCS mais utilizado atualmente, para começarmos a versionar os nossos projetos pessoais ou profissionais e como colocar o nosso projeto em uma plataforma de hospedagem de código com controle de versionamento como o Github.
